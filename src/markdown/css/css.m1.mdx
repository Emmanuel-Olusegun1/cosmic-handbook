# Getting Started with CSS

## Introduction to CSS

CSS (Cascading Style Sheets) is a styling language used to control the presentation of HTML elements. It allows developers to define colors, fonts, layouts, and spacing to enhance the appearance of a webpage.

## Why use CSS?

- Separates design from content for cleaner code.
- Improves user experience with structured layouts.
- Ensures consistency across multiple pages.
- Enables responsive design for all screen sizes.

## Recommended editors

- VS Code
- Sublime Text / Notepad

## Create your first CSS Project

### Step 1 : Set up files

- Create a folder (e.g., MyFirstWebsite).
- Inside, add two files:
  - index.html
  - styles.css

### Step 2 : Write Basic HTML

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First CSS Page</title>
    <link rel="stylesheet" href="styles.css"> <!-- Link to CSS -->
</head>
<body>
    <h1>Welcome to My Website</h1>
    <p>This is my first webpage with CSS.</p>
</body>
</html>
```

### Step 3 : Write Basic CSS

```css
body {
    background-color: lightblue;
    font-family: Arial, sans-serif;
}

h1 {
    color: darkblue;
    text-align: center;
}

p {
    color: darkgray;
    font-size: 18px;
}
```
