# Advanced HTML Elements and Usage

### Tables

- `<table> : Defines the table
- `<tr>` : Table row
- `<th>` : Table header
- `<td>` : Table data cell

###### Example : 

```html
<table border="1">
    <tr>
        <th>Name</th>
        <th>Age</th>
    </tr>
    <tr>
        <td>Alice</td>
        <td>25</td>
    </tr>
</table>

```

### Forms and Inputs

- `<form>` : Defines a form that collects Input
- `<label>` : Labels input fields
- `<input>` : Collects user input
- `<textarea>` : Multi-line input field
- `<button>` : Submits the form
- `<select>` : Creates dropdown options

###### Examples :

```html
<form action="submit.php" method="POST">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name" required>
    
    <label for="message">Message:</label>
    <textarea id="message" name="message"></textarea>

    <button type="submit">Submit</button>
</form>
```

### Semantic Elements

- `<header>` : Represents the top section of a webpage, often containing a logo or navigation.
- `<section>` : Groups related content together.
- `<article>` : Represents an independent, self contained piece of content like blog posts or news articles.
- `<aside>` : Contains side content, like sidebar or ads.
- `<footer>` : Represents bottom part of a webpage, usually containing copyright or contact info.


###### Example :

```html
<header>
    <h1>My Blog</h1>
    <nav>
        <a href="home.html">Home</a>
        <a href="about.html">About</a>
    </nav>
</header>

<section>
    <article>
        <h2>Understanding HTML</h2>
        <p>HTML is the foundation of web development...</p>
    </article>
</section>

<aside>
    <h3>Related Articles</h3>
    <ul>
        <li><a href="#">Learn CSS</a></li>
        <li><a href="#">JavaScript Basics</a></li>
    </ul>
</aside>

<footer>
    <p>&copy; 2025 My Blog. All rights reserved.</p>
</footer>
```

### Quotations and Citations

- `<blockquote>` : Displays a block-level quote, usually indented.
- `<q>` :  Displays inline quotes.
- `<cite>` : Cites a source or reference.

###### Example :

```html
<blockquote>
    "The best way to predict the future is to create it."
</blockquote>
<p>Albert Einstein once said, <q>Imagination is more important than knowledge.</q></p>
<p>This book was written by <cite>J.K. Rowling</cite>.</p>
```
