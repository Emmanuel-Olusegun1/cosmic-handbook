### Document Structure Elements

- <!DOCTYPE html> : Declares the document types as HTML5.
- <html> : The root element that wraps all content.
- <head> : Contains metadata, styles, and links to external files.
- <title> : Defines the page title.
- <body> : Contains the visible content of the page.

###### Example :

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <title>My First Page</title>
</head>
<body>
    <h1>Welcome to My Website</h1>
</body>
</html>
```

### Text Formatting Elements

| Tag          | Usage                |
| ------------ | -------------------- |
| <h1> to <h6> | Headings             |
| <p>          | Paragraphs           |
| <strong>     | Bold Text            |
| <em>         | Italic text          |
| <mark>       | Highlights text      |
| <u>          | Underlines text      |
| <s>          | Strinke through text |
| <sub>        | Subscript            |
| <sup>        | Superscript          |

###### Example :

```html
<h1>Main Heading</h1>
<p>This is a <strong>bold</strong> and <em>italic</em> text.</p>
<p><mark>Highlighted</mark> content.</p>
```

### Lists and Grouping Elements

- <ul> : unordered lists.
- <ol> : ordered lists.
- <li> : list item inside.
- <dl> : Description list
- <dt> : Term in a description list.
- <dd> : Description for the term.
- <div> : Block container for grouping content.
- <span> : Inline container for styling specific text.

###### Examples :

```html
<h2>Fruits</h2>
<ul>
    <li>Apple</li>
    <li>Banana</li>
    <li>Cherry</li>
</ul>
```

### Links and Navigation 

- <a> : Anchor Tags for creating hyperlinks to other pages or resources.
- <nav> : Navigation for Structuring primary navigation links.

### Images and multimedia

- <img> : Image tag for embedding images.
- <audio> and <video> : Incorporating multimedia content.
- controls : Adds play, pause, and volume controls.
